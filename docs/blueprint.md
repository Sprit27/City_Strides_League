# **App Name**: City Strides League

## Core Features:

- User Authentication: Secure user registration and login functionality.
- GPS Tracking: Utilize GPS to track running/walking speed and distance in real-time.
- Map Route Display: Display the current and recent running route on a map, without storing route data.
- Weekly Performance Tracking: Record and update the longest distance and fastest speeds (avg km/h and min/km) for each user weekly.
- League Board: Display leaderboards for distance, km/h, and min/km, updated weekly.
- Weekly Reset: Automatically reset user data (distance, km/h, min/km) to zero every Sunday.

## Style Guidelines:

- Primary color: Dark green (#386641) to evoke nature, health, and vitality.
- Background color: Light gray (#E9E5E3) to ensure readability and a clean, modern look.
- Accent color: Burnt orange (#A75D5D) for highlights and call-to-action elements, contrasting with the green to draw attention.
- Headline font: 'Space Grotesk' sans-serif, for headlines, for a modern and technical feel. Body font: 'Inter', also sans-serif.
- Use simple, modern icons to represent various data points (speed, distance, time) and user actions.
- A clean and intuitive layout with a focus on data visualization to display user progress and league standings.
- Subtle transitions and animations when updating data on the league board or user profile.
